Ankara genelinde hizmet veren profesyonel bir bahçe bakım işletmesi için geliştirilmiş; yüksek performanslı, estetik ve kullanıcı dostu bir web platformudur.

> **Slogan:** "Siz Hayal Edin, Biz Biçelim!"

---

## 🚀 Proje Hakkında

Bu proje, yerel bir işletmenin dijital dönüşümünü sağlamak amacıyla modern web teknolojileriyle inşa edilmiştir. Temel odak noktası, potansiyel müşterilere yapılan işleri en kaliteli şekilde sunmak ve hızlı bir iletişim kanalı oluşturmaktır.

- **Hız:** Vite ve React ile optimize edilmiş çalışma süresi.
- **UX:** Minimalist ve göz yormayan kullanıcı deneyimi.
- **Dönüşüm:** Belirgin "Call-to-Action" (Harekete Geçirici Mesaj) butonları.

## ✨ Öne Çıkan Özellikler

- **📸 Modern Galeri Mimarisi:** Videolar ve fotoğraflar için özel split-screen (masaüstü) ve mobil uyumlu kaydırılabilir interaktif galeri.
- **📱 Tam Responsive Tasarım:** Masaüstünden en dar mobil ekranlara (Samsung Galaxy A24 vb.) kadar kusursuz görünüm.
- **🔍 SEO Dostu:** Arama motorları için optimize edilmiş semantik HTML yapısı.
- **🎨 Kurumsal Kimlik:** Marka renklerine sadık, profesyonel tipografi ve ikon setleri.

## 🛠️ Teknik Yığın (Tech Stack)

| Araç                    | Açıklama                           |
| :---------------------- | :--------------------------------- |

## 💻 Kurulum ve Geliştirme

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1. **Depoyu klonlayın:**

   ```bash
   git clone [https://github.com/kullanici-adiniz/ankara-cim-bicme.git](https://github.com/kullanici-adiniz/ankara-cim-bicme.git)

   ```

2. **Proje dizinine gidin**

   ```bash
   cd ankara-cim-bicme

   ```

3. **Bağımlılıkları yükleyin**

   ```bash
   npm install

   ```

4. **Geliştirme sunucusunu başlatın**
   ```bash
   npm run dev
   ```

## 📂 Proje Klasör Yapısı

```text
├── src/
│   ├── components/    # Hero, Services, Portfolio, Features, Footer
│   ├── data/          # Sabit veriler (Hizmet listesi, Galeri verileri vb.)
│   ├── assets/        # Medya dosyaları (Görseller, Videolar)
│   ├── App.jsx        # Ana uygulama rotası ve bileşen birleşimi
│   └── index.css      # Tailwind konfigürasyonu ve global stiller
├── public/            # Statik dosyalar (favicon, manifest vb.)
└── package.json       # Bağımlılıklar ve script tanımları
```
